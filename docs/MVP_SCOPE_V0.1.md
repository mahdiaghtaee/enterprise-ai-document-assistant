# MVP Scope - Version 0.1.0

## Objective
Deliver a working end-to-end demonstration suitable for technical interviews and client presentations.

## Included
- Document upload
- Document metadata storage
- Background ingestion pipeline
- Basic text extraction
- Semantic retrieval
- Question answering with document references
- Docker Compose deployment
- API documentation

## Deferred
- OCR
- Multi-tenant implementation
- Advanced permissions
- Admin dashboard
- Multiple LLM providers

## Success Criteria
A new developer can clone the repository, start the services with Docker Compose, upload a sample document, ask a question, and receive an answer with source references.
