# Code of Conduct

## Our Pledge

We want this project to be a respectful and productive place for learning, collaboration, and practical software development.

## Expected Behavior

Participants are expected to:

- Be respectful and constructive.
- Focus feedback on ideas, code, documentation, and project direction.
- Keep discussions relevant to the project.
- Help create a welcoming environment for contributors with different experience levels.

## Unacceptable Behavior

Unacceptable behavior includes harassment, personal attacks, discriminatory language, or repeated disruptive conduct.

## Reporting

If you notice behavior that does not fit this code of conduct, open a private conversation with the project maintainer through GitHub profile contact options.

## Enforcement

The maintainer may edit, hide, lock, or remove content that is not aligned with this code of conduct. Repeated violations may lead to removal from project participation.
