# Enterprise AI Document Assistant

Enterprise AI Document Assistant is a production-oriented backend project that demonstrates how to build a secure AI-powered document assistant for businesses.

The project combines ASP.NET Core and Python (FastAPI) to process documents, manage metadata, and prepare the foundation for Retrieval-Augmented Generation (RAG).

## Business Problem

Organizations store valuable knowledge inside PDFs, contracts, reports, policies and internal documentation. This project provides the foundation for a private AI assistant capable of understanding those documents.
